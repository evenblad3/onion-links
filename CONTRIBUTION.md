# CONTRIBUTION
I am very thankful for showing your interest in improving this repository.

## Formatting
| Site | Description | NSFW |
|:---|:---|:---|
| Site Name (Link to the site) | Short and neutral description | Is it NSFW? (Yes/No) |

Example entry:
```markdown
| [Facebook](http://facebookwkhpilnemxj7asaniu7vnjjbiltxjqhye3mhbshg7kx5tfyd.onion) | Social networking site to connect with friends online | No |
```

## Pull Request Guidelines
* Do **NOT** submit illegal, scam, harmful sites. 
* Avoid submitting v2 onion links.
* Make sure to label NSFW sites
* Never submit a site that is listed already, unless the site address is officially changed.
* Avoid trailing forward slashes (`/`) in links.
* Follow the alphabetical order
* Follow the [formatting rules](#formatting)
* Have a short and clear commit message.
